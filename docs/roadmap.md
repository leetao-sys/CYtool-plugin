# Roadmap

## Milestone 1: Requirements And Design

- Finalize requirements.
- Confirm frontend stack.
- Confirm plugin runtime boundary.
- Confirm SSH escalation scope.

## Milestone 2: Platform MVP

- Backend FastAPI skeleton.
- SQLite metadata database.
- Plugin package validation.
- Plugin install/list/enable/disable/uninstall/update APIs.
- Frontend management UI.

## Milestone 3: Plugin Runtime

- Sidebar menu from enabled plugins.
- Plugin page loading.
- Plugin asset serving.
- Operation logs.

## Milestone 4: Platform APIs

- SQLite/local database API.
- Remote database adapter interface.
- SSH command execution.
- SFTP upload/download.
- Batch SSH execution.
- Root escalation model.

## Milestone 5: Reference Plugins

- JSON formatter.
- Time converter.
- Encoding converter.
- Remote command executor.

## Milestone 6: Validation And Packaging

- Automated test coverage.
- Manual smoke test.
- Package reference plugin zips.
- Evaluate `.exe` packaging size.
